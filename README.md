Installation
============

Install RabbitMQ server

```shell
brew install rabbitmq
```

Start RabbitMQ server

```shell
rabbitmq-server
```

Install Rubinius

```shell
rbenv install rbx-2.2.1
rbenv local rbx-2.2.1
```

Install bundle

```shell
gem install bundle
bundle install
```

Run producer

```shell
bin/producer
```

Run consumer

```shell
bin/consumer
```
